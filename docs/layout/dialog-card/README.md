## 使用方法
使用 dialog-card 组件，由于使用场景较多，这个组件已经全局注册，直接使用 `<dialog-card></dialog-card>` 即可。
组件默认没有内边距，

```
<dialog-card title="会员资料修改履历">
  <div class="p10"></div>
</dialog-card>
 ```
